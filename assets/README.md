# Assets

Shared media assets used by the vault.

## Structure

- `images/`: embedded screenshots and pasted images used by Markdown notes.

## Naming Policy

- Pasted images use `pasted_YYYYMMDDHHMMSS.png`.
- Imported WhatsApp images use `whatsapp_YYYYMMDD_HHMMSS_identifier.jpg`.
- Keep image filenames stable because Obsidian embeds depend on exact names.
